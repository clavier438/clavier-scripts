## Phase 1 — OCI 부트 (실행 + 검증, ~10분)

`oci-connect` 로 OCI 접속 후:

```bash
# OCI 서버 안에서
git clone https://github.com/clavier0/clavier-scripts ~/clavier-scripts
cd ~/clavier-scripts && git checkout claude/fix-local-commit-issues-6XdAx
bash clouds/oci/bootstrap-agent.sh        # apt + Node + Doppler + Claude + sibling clone
doppler login                              # interactive (Mac 으로 browser open 됨)
doppler setup --project clavier --config prd
```

**검증 명령** (전부 OCI 안에서):

```bash
# 1) sibling clone 성공
ls ~ | grep -E '^(clavier-hq|platform-workers|clavier-scripts)$'
# 기대: 3줄

# 2) Claude Code 작동
doppler run -- claude --version

# 3) workerCtl sibling-first (Cloudflare 토큰 실제 호출)
doppler run -- node ~/clavier-scripts/tools/workerCtl.mjs panel

# 4) doc-coverage sibling-first
bash ~/clavier-scripts/tools/doc-coverage.sh --recent
```

**실패 처리**:
- `apt-get` 실패 → 사용자에게 sudo 권한 확인
- `doppler login` 거부 → URL 사용자 모바일/Mac 브라우저로 열어서 인증
- `workerCtl panel` Cloudflare 401 → CFAT 토큰 Doppler 등록 확인
- 그 외 OCI-specific 이슈 → web 세션이 만든 가정 깨졌다는 신호. 정확히 어디서 어떻게 실패했는지 conversation 으로 보고 → 별 commit 으로 fix.

성공 시 → Phase 2.
