# Layer 1 — Core Principles (불변 원리)

> 브랜드 카피 어떤 프로젝트에서도 항상 지킨다. 4원칙.
> 이 4원칙이 곧 **기본 인격 기준점**이다. Layer 2(브랜드 스타일)는 이 인격 위에서 상황별 스펙트럼으로 변주된다.

---

## 원칙 1. 단어로 결과를 박지 말고, 매혹적 진실의 힌트와 근거로 흘려라

> 형용사·자기 자랑·손님 감정 통제·경쟁 우위 주장 — 이 모든 것은 단어로 독자 머릿속에 결과를 직접 박으려는 행위다. 독자는 슬로건과 형용사로 설득되지 않는다.
>
> 대신 확인 가능한 사실, 구체적 출처, 동사, 숫자, 역사로 장면과 근거를 흘려라. 시설이나 제품을 나열하기보다는, 그것이 독자에게 어떤 경험과 영향을 주는지를 구체적인 메커니즘으로 보여라. 때로는 드러나는 한계를 솔직하게 인정함으로써 오히려 나머지 주장의 신뢰를 높이기도 한다.

| 무엇을 표현하고 싶은가 | 이렇게 표현하지 말고 | 이렇게 표현해라 (무카유) |
|---|---|---|
| 컨셉 ("자연 친화") | 자연과 함께하는 힐링 공간 | 아카마츠나 산벚나무, 수양벚나무, 여름 동백나무, 단풍, 동백을 비롯해 느긋하게 번창한 수목을 볼 수 있습니다 |
| 재료 품질 ("신선") | 신선한 게 요리 | 가가 하시다테 항에서 들어온 털게 |
| 분위기 ("아늑") | 아늑하고 평온한 객실 | 봄은 산벚꽃의 꽃잎이, 가을은 단풍이 춤추어 오는 특별실입니다 |
| 시설 우수성 | 최고급 시설을 자랑합니다 | 노천탕이 있는 일본식 방 4개. 산정 전체를 바라볼 수 있는 달 전망대(테라스)가 있습니다 |
| 메뉴 풍성함 | 봄 코스 메뉴 | 선착 — 여름 야채 / 가가 하시다테 항에서 온 털게 앰버 젤리 / 가가 태후의 스프 (시오 마사리) |
| 공간 크기 | 넓은 거실 | 거실에는 길이 3m의 닦아 옻칠의 테이블을 갖추고 있습니다 |
| 공간 역사·전통 | 오래된 료칸의 전통 | 가가시의 2개의 하천(다이쇼지강·도바시강)이 다른 시정촌에 걸쳐 있지 않고, 수원으로부터 하구까지 가가시 안에서만 완결되어 있습니다. 해발 1368m 대일산이 원류입니다 |
| 계절감 | 가을의 풍성한 식탁 | 도야마쵸가 소침 — 젓가락 시즈코·덩굴콩·적피 호박·금시초·고로시마 금시 |
| 이름·고유성 | 와카무라사키 스위트 | 와카유키 스페셜룸 |
| 환대·정성 | 저희는 정성껏 모시겠습니다 | 다도는 료칸 주인 부부가 직접 안내합니다 |
| 식재료 우수성 | 우리 식재료는 최고급입니다 | 가가시 안에 맹금류 11종이 삽니다. 전국에서 볼 수 있는 맹금류의 70%입니다. 생태계의 다양성이 그대로 음식의 다양성으로 연결됩니다 |
| 경쟁 우위·우월함 | 타의 추종을 불허하는 료칸 | 총 16개의 객실. 모든 객실에는 샤워 부스와 온천 노천탕이 있습니다 |
| 손님 감정 통제 | 잊지 못할 추억을 만드세요 | 봄은 산벚꽃의 꽃잎이, 가을은 단풍이 춤추어 오는 특별실입니다 |
| 셰프·권위 | 우리 셰프는 미슐랭 출신입니다 | 상어의 부드러운 삶은 — 요리장의 스페셜티 |
| 신뢰감 (고가·신중한 결정) | 신선한 카가의 식재료 | 가가시의 2개의 하천(다이쇼지강·도바시강)이 다른 시정촌에 걸쳐 있지 않고, 수원에서 하구까지 가가시 안에서만 완결됩니다 |
| 주체 (저희·우리) | 저희는 ~을 제공합니다 | 가가시의 2개의 하천이 이 고장에서 바로 바다로 흐릅니다 |
| 한계·약점 | (회피 / 모호하게 처리) | 입구에 18단의 계단이 있습니다 |

## 원칙 2. 시설·제품을 나열하지 말고, 독자에게 일어나는 일을 설명한다

> 시설이나 제품의 존재를 나열하기보다는, 그것이 사람에게 어떤 경험과 영향을 미치는지를 구체적으로 보여주는 편이 더 자연스럽다.
>
> 구체적인 스펙(크기, 구성, 시설 목록 등)은 본문에서 과도하게 풀어 설명하지 말고, 별도의 리스트나 구조로 분리하는 편이 좋다.

| 카테고리 | 이렇게 표현하지 말고 | 이렇게 표현해라 |
|---|---|---|
| 수면 | 킹사이즈 침대와 고급 매트리스를 갖추고 있습니다 | 침실에는 세미 더블 사이즈 침대 (120×195cm) 2대를 갖추고 있습니다 |
| 공간 크기 | 가장 작은 방은 너비 2.1m입니다 | 방의 넓이는 70㎡입니다 |
| 창문 / 전망 | 모든 객실 창문이 정원을 향해 있습니다 | 산정 전체를 바라볼 수 있는 달 전망대(테라스)가 있습니다 |
| 자연과의 관계 | 숲 전망이 좋은 객실입니다 | 수령 100년을 넘는 산벚나무의 대목을 정면으로 바라봅니다 |
| 공간 경험 | 넓은 거실이 있습니다 | 거실에는 길이 3m의 닦아 옻칠의 테이블을 갖추고 있습니다 |
| 비품 | 다양한 어메니티가 준비되어 있습니다 | 샴푸·컨디셔너·바디워시·바디로션·핸드워시·클렌징젤·페이셜워시·로션·에멀젼 |
| 온천 / 목욕 | 객실에 노천탕이 있습니다 | 히노키의 온천 노천탕. 모든 객실에 샤워 부스와 온천 노천탕이 있습니다 |

**스펙 분리 예시**

```
넓은 70㎡ 공간에 자쿠지와 다도 세트가 완비되어 있으며, 최대 3인까지 수용 가능합니다.
```
↓
```
◎ 방의 넓이는 70㎡입니다
◎ 정원 1~3인
◎ 자쿠지 · 다도 세트
◎ 히노키 온천 노천탕
```

## 원칙 3. 기본 말투를 유지한다

> 무카유(베니야 무카유) 실제 카피가 기준이다. 사실·숫자·고유명사·동사. 판단은 독자에게 넘긴다.

**무카유 실제 문장 — 이것이 기준**

```
가가시의 2개의 하천(다이쇼지강·도바시강)이 다른 시정촌에 걸쳐 있지 않고,
수원으로부터 하구까지 가가시 안에서만 완결되어 있습니다.
해발 1368m 대일산이 원류입니다.

가가시 안에 맹금류 11종이 삽니다. 전국에서 볼 수 있는 맹금류의 70%입니다.

수령 100년을 넘는 산벚나무의 대목을 정면으로 바라봅니다.
봄은 산벚꽃의 꽃잎이, 가을은 단풍이 춤추어 오는 특별실입니다.

◎ 방의 넓이는 120㎡입니다
◎ 침실에는 세미 더블 사이즈 침대 (120×195cm) 2대
◎ 워크인 클로젯

鮑 구이 — 간장에서 먹는 鮑 구이.
◎ 4월 28일~9월 중순의 기간 한정
◎ 예약 필요
◎ 1접시 14,800엔 (소비세 10%와 서비스료 10% 포함)

거실에는 길이 3m의 닦아 옻칠의 테이블을 갖추고 있습니다.
```

**리듬 패턴 — 이렇게 섞는다**

| 블록 | 형태 | 예 |
|---|---|---|
| 서술 단락 | ~있습니다 / ~됩니다 / ~바라봅니다 | "수원에서 하구까지 가가시 안에서만 완결되어 있습니다." |
| 스펙 나열 | ◎ 명사 | ◎ 120㎡ ◎ 노천탕 ◎ 예약 필요 |
| 짧은 사실 | 명사 마감 / 숫자 마감 | "전국의 70%입니다." "3m 옻칠 테이블." |
| 계절·기간 한정 | ◎ 날짜 | ◎ 4월 28일~9월 중순의 기간 한정 |

## 원칙 4. 모든 것은 하나의 Big Idea로 정렬된다

> 한 프로젝트에는 지배 아이디어가 하나다. 그것이 Big Idea다. 모든 섹션·모든 문장은 Big Idea를 다른 각도에서 증명한다. 방향은 하나여야 한다.

**카피 작업 전에 반드시 한다**
이 브랜드의 Big Idea가 무엇인지 먼저 확인한다. 명시되지 않았다면 브랜드 자료에서 추출하거나, 클라이언트에게 물어 합의한 뒤 시작한다. Big Idea를 모른 채 섹션별로 쓰기 시작하지 않는다.

**쓰면서 이 신호를 주의한다**

| 경고 신호 | 예 |
|---|---|
| 섹션마다 내세우는 가치가 달라진다 | 객실 섹션은 "자연·고요"를 강조하다가, 식당 섹션에서 갑자기 "럭셔리·미슐랭 권위"로 전환된다 |
| Big Idea와 무관한 좋은 사실을 끼워 넣는다 | 사실이 맞더라도 Big Idea를 향하지 않으면 삭제하거나, 방향을 바꿔 재구성한다 |
| 좋은 문장들이 모여 있지만 인상이 흩어진다 | 원칙 1·2를 모두 지켜도, 정렬 없이 나열된 카피는 독자 머릿속에 아무것도 남기지 못한다 |

→ Big Idea를 어떻게 구체화하고 펼치는지는 Layer 2 (브랜드 스타일).

## 4원칙 요약

| | 원칙 | 차원 |
|---|---|---|
| **1** | 단어로 결과를 박지 말고, 매혹적 진실의 힌트와 근거로 흘려라 | 논리·전략 |
| **2** | 시설·제품을 나열하지 말고, 독자에게 일어나는 일을 설명한다 | 소재·접근 |
| **3** | 기본 말투를 유지한다 — 무카유 실제 문장이 기준 | 어체·인격 |
| **4** | 모든 것은 하나의 Big Idea로 정렬된다 | 정렬·메타 |

→ **원칙 1·2·3 = 기본 인격 기준점.** 어떤 브랜드든 깨지 않는다.
→ **원칙 4 = 그 인격 위에서 한 브랜드가 한 정신으로 수렴되는 메타.**
→ **Layer 2 = 그 인격·정신 위에서 브랜드별 스펙트럼으로 변주.** 제로베이스에서 시작하지 않는다.
