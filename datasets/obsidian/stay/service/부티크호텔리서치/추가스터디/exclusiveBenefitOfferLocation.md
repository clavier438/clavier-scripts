# 배치 원칙

**1. 니치가 극단적 (건축, 가족 전용):** → **나중에 배치** (네 번째) → 이유: 니치 자체가 Exclusive, 감정적 투자 우선 → 사례: Cavallino (100%), Borgo Santo Pietro (75%)

**2. 니치 경쟁 있음 (로맨틱, 웰니스):** → **앞에 배치** (두 번째) → 이유: OTA 비교 전 차별화 필요 → 사례: Burrus House (80-85%), Il Sereno (60-80%)

**3. 항상 중요한 것:** → **Sticky CTA "Book Direct - Best Rates"** → 배치와 무관하게 항상 가시성 → 모든 극강 사례 100% 적용

#### 전략 A: 극단적 니치 (Cavallino 유형)

**니치가 강할 때 = Exclusive는 나중에**

```
Home (니치 스토리텔링)
├── Philosophy/About ← OTA가 설명 못하는 것
│   (예: 건축 철학, 공간의 의미)
├── Rooms ← 니치 맞춤 디자인
├── Experiences ← 독점 경험
│   (예: 건축 투어, 오너 대담)
├── Offers ← 보너스처럼
│   (예: 건축 워크숍 할인)
└── Book Direct
```

**이유:**

- 웰니스/로맨틱 = OTA에도 많음
- 가격 비교 전 차별화 필요
- "여기만" 메시지 선점

**예상 효과:**

- OTA 이탈 방지
- 전환율 4-6%

#### 전략 B: 경쟁 있는 니치 (Burrus 유형)

**니치 경쟁 있을 때 = Exclusive는 앞에**

```
Home (Hero)
├── Exclusive Packages ← 두 번째
│   ├── OTA 미제공 강조
│   ├── 직접 예약 혜택
│   └── 한정 수량
├── Rooms
├── Experiences
└── Book Direct (Best Rates)
```

**이유:**

- 건축 애호가 = Blue Ocean
- OTA 검색에 나올 수 없음
- Philosophy 자체가 Exclusive

**예상 효과:**

- 감정적 투자 극대화
- Offers는 "결정 밀어주는 마지막 푸시"
- 전환율 5-8%
