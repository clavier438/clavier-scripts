# 조사 방법론: "결과"가 아니라 "변곡점 사이의 델타"를 찾아라

## 배경 — The Lost Kitchen 케이스에서 얻은 교훈

The Lost Kitchen(메인주 Freedom, 인구 719명)의 예약 전화가 2016년 26건 → 2017년 10,000건으로 **384배** 폭증했다. 이 현상을 조사하면서 범한 실수와 그로부터 도출한 방법론을 정리한다.

### 범한 실수

- "전화 10,000건이 쏟아졌다"는 \*\*현상(결과)\*\*을 마치 원인인 것처럼 서술했다.
- "수년간의 노력과 희생", "입소문의 복리 효과"라는 **동어반복**으로 설명을 대체했다.
- 실제 원인인 3가지 구체적 트리거를 찾아놓고도 **"다만"이라는 접속사로 부차적 정보 취급**했다.

### 실제 원인 (찾았지만 묻어버린 것들)

| 시점 | 사건 | 효과 |
| --- | --- | --- |
| 2015년 6월 | Michael Dutton(미디어 프로듀서) 합류 | 미디어 전략·PR 관리 시작 |
| 2016년 초 | James Beard 세미파이널리스트 선정 | 전국적 요리계 인지도 획득 |
| 2016년 중 | Tastemade × L.L.Bean 숏필름 공개 | 200만 뷰, 온라인 전국 노출 |
| 2016년 중 | L.L.Bean 브랜드 앰배서더 계약 | 기존 아웃도어 팬층에 노출 |
| 2016년 중 | Martha Stewart, Food & Wine 등 피처 | 전통 미디어 인지도 |

이 다섯 가지가 2016년 4월\~2017년 3월 사이에 **새로 추가된 변수**였고, 이전 조건(노력, 맛, 로컬 입소문)은 2014\~2015에도 동일하게 존재했다. 따라서 폭발의 원인은 이것들뿐이다.

---

## 방법론 4단계 (요약)

### 1단계 — 결과를 원인으로 착각하지 마라

"전화가 터졌다", "입소문이 퍼졌다", "복리 효과가 쌓였다"는 전부 **현상의 재서술**이지 설명이 아니다.

**체크 방법:** 내 설명에서 결과 표현을 빼면 아무 정보도 남지 않는다면, 그건 설명이 아니라 동어반복이다.

### 2단계 — Before/After 사이에 "뭐가 바뀌었는지" 찾아라

- ❌ "왜 터졌어?"
- ✅ "Before(26건)와 After(10,000건) 사이에 **구체적으로 뭐가 달라졌어?**"

**새로 추가되거나 변경된 변수만이 후보**다.

### 3단계 — 구체적인 이름·날짜·숫자를 찾아라

이름과 숫자가 없으면 아직 원인을 찾은 게 아니다.

### 4단계 — 찾은 걸 "다만"으로 묻지 마라

**델타를 설명하는 구체적 트리거가 메인. 나머지가 부연.**

---

## 실행 프레임워크: Delta Trigger Analysis (DTA)

### 이론적 기반

존 스튜어트 밀의 \*\*차이법(Method of Difference)\*\*을 실무 조사에 적용한 것이다.

> "현상이 발생한 경우(After)와 발생하지 않은 경우(Before)의 조건이 모두 동일한데 하나만 다르면, 그 하나가 원인이다."

Before에도 있었던 조건은 전부 \*\*상수(constant)\*\*로 제거하고, After에만 존재하는 \*\*새 변수(new variable)\*\*만 원인 후보로 남긴다.

---

### Phase 1: 현상 정량화

| 항목 | 작성 내용 |
| --- | --- |
| **Before 수치** | 구체적 숫자 + 시점 (예: 2016년 4월, 예약 26건) |
| **After 수치** | 구체적 숫자 + 시점 (예: 2017년 4월, 예약 10,000건) |
| **배율** | After ÷ Before (예: 384배) |
| **델타 기간** | Before → After 사이 구간 (예: 2016.04 \~ 2017.03, 12개월) |

**체크:** Before와 After가 둘 다 숫자로 표현되지 않으면 이 프레임워크를 적용할 수 없다. 먼저 정량화하라.

---

### Phase 2: 상수 제거

델타 기간 이전에도 존재했던 조건을 전부 나열하고, **원인 후보에서 제거**한다.

| 상수 후보 | Before에도 있었나? | 제거 여부 |
| --- | --- | --- |
| Erin French의 요리 실력 | ✅ 2014\~부터 동일 | ❌ 제거 |
| Freedom 소읍 위치의 희소성 | ✅ 동일 | ❌ 제거 |
| 로컬 입소문 | ✅ 동일 | ❌ 제거 |
| "수년간의 노력과 희생" | ✅ 동일 | ❌ 제거 |

**규칙:** "Before에도 있었는데 Before에서는 안 터졌다"면, 그건 원인이 아니다.

**주의 — 상수처럼 보이는 변수:**\
일부 조건은 Before에도 존재했지만 **규모나 성격이 질적으로 달라진** 경우가 있다. 예를 들어 "입소문"이 Before에는 Freedom 반경 50km였는데 After에는 전국이었다면, 그 **확산 범위의 변화를 유발한 트리거**가 진짜 원인이다. 입소문 자체가 아니라.

---

### Phase 3: 새 변수 탐색

델타 기간에 **새로 추가된 변수**를 아래 6개 카테고리에서 찾는다.

#### 카테고리 1: 사람 (People)

새로 합류한 팀원, 어드바이저, 파트너, 투자자.

| 찾을 것 | 예시 |
| --- | --- |
| 이름 | Michael Dutton |
| 직함/역할 | 미디어 프로듀서 |
| 합류 시점 | 2015년 6월 |
| 이전 경력 | Broadway Video, VH1, AP통신 |
| 기여 메커니즘 | 미디어 전략 설계, 브랜드 딜 중개, PR 관리 |

#### 카테고리 2: 콘텐츠 (Content)

새로 제작/공개된 영상, 기사, 소셜 포스트.

| 찾을 것 | 예시 |
| --- | --- |
| 제목 | Tastemade "Uncharted" 시리즈 |
| 플랫폼 | Facebook, Tastemade |
| 공개 시점 | 2016년 중 |
| 도달 규모 | 200만 뷰, 19,000 좋아요, 4,000 공유 |
| 스폰서/제작 주체 | L.L.Bean 공동 제작 |

#### 카테고리 3: 제도적 인정 (Institutional Recognition)

어워드, 랭킹, 인증, 세미파이널리스트 선정.

| 찾을 것 | 예시 |
| --- | --- |
| 어워드명 | James Beard Award |
| 카테고리 | Best Chefs in America |
| 선정 단계 | 세미파이널리스트 |
| 선정 시점 | 2016년 초 |
| 업계 영향력 | 세미파이널리스트 96%가 사업 증가 보고 |

#### 카테고리 4: 비즈니스 딜 (Business Deals)

브랜드 파트너십, 앰배서더 계약, 유통 계약, 라이선스.

| 찾을 것    | 예시                     |
| ------- | ---------------------- |
| 브랜드명    | L.L.Bean               |
| 계약 유형   | 브랜드 앰배서더               |
| 계약 시점   | 2016년 중                |
| 노출 메커니즘 | 기존 아웃도어 팬층(수백만)에 교차 노출 |

#### 카테고리 5: 플랫폼/알고리즘 변화 (Platform Shifts)

소셜 미디어 알고리즘 변경, 새 플랫폼 등장, 검색 엔진 변화. 해당 없으면 비워둔다. 억지로 채우지 않는다.

#### 카테고리 6: 외부 맥락 (External Context)

문화 트렌드, 경쟁자 실패, 규제 변화, 시즌 이벤트. 직접 트리거가 아닌 배경 조건은 보조 요인으로만 기재한다.

---

### Phase 4: 인과 메커니즘 검증

새 변수를 찾았으면, 각각에 대해 **"이것이 Before → After 변화를 물리적으로 어떻게 만들었는지"** 메커니즘을 서술한다.

| 새 변수 | 인과 메커니즘 | Before에 없었나? | 규모 충분한가? |
| --- | --- | --- | --- |
| Dutton 합류 | 미디어 전략 설계 → 콘텐츠/딜 세팅 가능 | ✅ | ✅ 아래 2, 3의 전제 조건 |
| Tastemade 영상 | 200만 명에게 시각적 노출 → 인지도 전국 확대 | ✅ | ✅ 200만 뷰 |
| James Beard 선정 | 요리계 권위 인증 → 미디어/셰프 네트워크 주목 | ✅ | ✅ 업계 최고 권위 |
| L.L.Bean 딜 | 아웃도어 팬층 교차 노출 | ✅ | ✅ 고객 수백만 |

**탈락 기준:**

- Before에도 있었으면 → 탈락
- 메커니즘을 서술할 수 없으면 → 탈락 (상관관계일 뿐)
- 규모가 Before→After 배율을 설명하기엔 부족하면 → 보조 요인으로 격하

---

### Phase 5: 답안 작성 규칙

#### 구조

```
[1] 리드: 가장 강력한 새 변수 1~3개로 시작 (이름/숫자/날짜 포함)
[2] 메커니즘: 각 변수가 어떻게 결과를 만들었는지 서술
[3] 배경: 상수(노력, 실력, 입지)는 여기서 간략히 언급
```

#### 금지 표현

| 금지 | 이유 |
| --- | --- |
| "다만 그 사이에 \~도 있었다" | 트리거를 부차적으로 격하시킴 |
| "복리 효과가 쌓였다" | 메커니즘 없는 동어반복 |
| "입소문이 퍼졌다" | 결과의 재서술 |
| "수년간의 노력으로" | Before에도 동일했던 상수 |
| "우연히 터졌다" | 구체적 트리거를 못 찾았다는 고백 |

#### 작성 예시

❌ **나쁜 답안:**\
"Erin French가 수년간의 노력과 희생으로 쌓아올린 것입니다. 다만 그 사이에 James Beard House 초청 요리, 로컬 입소문의 복리 효과가 임계점을 넘긴 것으로 보입니다."

✅ **좋은 답안:**\
"2015년 미디어 프로듀서 Michael Dutton이 합류하면서 전략적 미디어 세팅이 시작됐다. 2016년 Tastemade × L.L.Bean 숏필름(200만 뷰), James Beard 세미파이널리스트 선정, L.L.Bean 앰배서더 계약이 12개월 안에 동시 집행되면서 메인주 로컬 식당이 전국적 인지도를 확보했다. 2017년 예약 시즌에 이 노출의 총합이 한꺼번에 유입되면서 26건 → 10,000건(384배)으로 폭증했다."

## DTA 체크리스트

- [ ] Phase 1: Before/After를 숫자로 정량화했는가?

- [ ] Phase 2: Before에도 있던 상수를 전부 제거했는가?

- [ ] Phase 3: 6개 카테고리에서 새 변수를 탐색했는가?

- [ ] Phase 4: 각 새 변수의 인과 메커니즘을 서술했는가?

- [ ] Phase 5: 트리거를 리드로, 상수를 배경으로 배치했는가?

- [ ] Phase 5: 금지 표현을 사용하지 않았는가?

---

## 요약 공식

> **"왜 터졌어?"** → \*\*"뭐가 바뀌었어?"\*\*로 질문을 바꾸고,\
> 바뀐 것의 **구체적 이름·날짜·숫자**를 찾아라.\
> 찾았으면 그게 답의 **중심**이다. 절대 "다만"으로 밀어내지 마라.
