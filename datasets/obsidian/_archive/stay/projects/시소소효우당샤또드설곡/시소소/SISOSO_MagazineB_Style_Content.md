### 2. Introduction / About

```
강원도의 끝자락에 시소소가 있습니다.

굽이진 길을 따라 한참을 달려야 하는 곳.
마지막 언덕을 넘으면 삼척 바다가 모습을 드러냅니다.

이곳은 기다림 끝의 기쁨을 위한 공간입니다.

입구에 들어서는 순간부터 
바다를 향한 기대가 천천히 차오릅니다.

객실 안, 커튼을 걷는 순간
탁 트인 동해가 시야를 가득 채웁니다.

멀고 고요한 여정을 마친 이들에게
온전한 쉼을 건넵니다.

바다를 향한 기대와 마주함의 기쁨.
이곳에서 천천히 느껴보세요.
```

### 3. Philosophy (공간 철학)

```
리셉션까지의 거리

시소소는 입구에서 곧장 리셉션을 마주하지 않습니다.
조금 더 들어가야 합니다.
조용히 호스트를 만나기까지의 짧은 여정.

불필요한 동선을 덜어낸 리셉션에서는
오직 차 한 잔에 집중합니다.
일상에서 한 발 떨어진 감각을 경험하게 됩니다.

---

전통과 현대 사이

시소소의 객실은 모두 바다를 향해 열려 있습니다.
평상, 기둥, 지붕의 형태가 어우러진 공간.
전통의 형식을 따라 현대적 감각으로 다듬었습니다.

고요한 시간을 함께하는 방식입니다.
```

### 4. Unique Features (3가지 특징)

#### Feature 1

```
물의 휴식

삼척의 600년 전통주 '불술'.
오직 시소소에서만 만나는 '청곡주'와 '삼수령주'.

잔잔하면서도 힘 있는 삼척 바다를 바라보며 
긴 호흡으로 즐기는 차.

시소소와 삼척의 거친 바다 향기를 담은 세 가지 향.

마지막으로, 몸을 푹 담글 수 있는 자쿠지.

다양한 형태의 물이 주는 휴식입니다.
```

#### Feature 2

```
600년 전통의 '온' 차림

하루 네 팀만 이용할 수 있는 저녁 식사.

해풍을 맞으며 직접 키운 신선한 채소들.
정성스럽게 만든 소스.
매 순간을 소중하게 채우고자 하는 마음.

갓 지은 문어 솥밥과 한돈 안심구이.
삼척의 식재료로 건강한 배부름을 전합니다.

박병준 명장의 600년 전통 토속주 '불술'과
삼척에서만 맛볼 수 있는 술들.
```

#### Feature 3

```
세 가지 객실

정온재(靜溫齋)
나만의 고요한 시간을 위한 곳입니다.

청차재(靑茶齋)
푸른 바다와 하늘 아래 차를 즐기는 곳입니다.

화온재(和溫齋)
가족과 함께하는 곳입니다.
가족탕과 사우나가 있습니다.

여행 스타일에 맞는 공간을 선택하세요.
```
