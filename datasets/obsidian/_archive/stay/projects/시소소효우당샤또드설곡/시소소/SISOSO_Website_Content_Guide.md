## Instagram 추가 수집 가이드

### 왜 Instagram도 수집해야 하나?
- UGC (User Generated Content) 확보
- 실제 투숙객 경험 이미지
- 계절별/시간대별 다양한 무드 컷
- SNS 감성에 맞는 컨텐츠

### Apify로 Instagram 수집하기

#### 1단계: Apify 계정 생성
- https://apify.com 가입
- 무료 플랜: 월 $5 크레딧 제공

#### 2단계: Instagram Scraper 선택
- Actor: `apify/instagram-scraper`
- URL: https://apify.com/apify/instagram-scraper

#### 3단계: 설정 입력
```json
{
  "username": ["stay_sisoso"],
  "resultsLimit": 100,
  "searchType": "user",
  "searchLimit": 100
}
```

#### 4단계: 실행 및 다운로드
- Run 버튼 클릭
- 5-10분 대기
- Dataset → Export → JSON 다운로드

#### 5단계: 이미지 정리
다운로드한 JSON에서:
- `displayUrl`: 고해상도 이미지 URL
- `caption`: 캡션 텍스트
- `timestamp`: 게시 날짜
- `likesCount`: 좋아요 수

### 예상 비용
- 포스트 100개 수집: 약 $0.50-1
- 월 5-6개 호텔 수집 가능 (무료 크레딧 내)
