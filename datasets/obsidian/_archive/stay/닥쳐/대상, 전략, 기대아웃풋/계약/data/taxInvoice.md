세금계산서 발행
전제조건
	•	사업자등록증이 있어야 세금계산서 발행 가능
	•	개인사업자 또는 법인사업자 모두 가능
	•	사업자등록 안 되어 있으면 → 먼저 세무서 방문해서 사업자등록 필수
세금계산서란?
	•	사업자가 거래 시 부가가치세(VAT 10%)를 별도로 받고, 이를 국세청에 신고하기 위한 공식 증빙서류
	•	클라이언트(매입자)는 이 세금계산서로 부가세 공제 가능
	•	법적으로 거래 후 익월 10일까지 발행 의무


방법 1: 홈택스(국세청 공식, 무료) — 추천
	1.	홈택스 접속 (https://hometax.go.kr)
	2.	공동인증서 또는 간편인증(카카오, 네이버 등)으로 로그인
	3.	전자세금계산서 → 전자세금계산서 발급 클릭
	4.	아래 정보 입력:
	•	공급자(당신): 사업자등록번호, 상호, 주소
	•	공급받는자(클라이언트): 사업자등록번호, 상호, 주소
	•	작성일자: 입금받은 날짜 또는 계약일
	•	공급가액: 250만원 (부가세 별도)
	•	세액: 25만원 (공급가액의 10%)
	•	합계금액: 275만원
	•	품목: “웹사이트 제작 용역”
	5.	발행 클릭 → 국세청에 자동 전송, 클라이언트 이메일로도 전송됨
방법 2: 세무 프로그램 (유료/무료)
	•	더존 SmartA, 세무사랑, 비즈넷, 팝빌 등
	•	월 1~2만원, 대량 발행/관리 편리
	•	홈택스보다 UI 편리, 자동 알림/보관 기능
방법 3: 세무사 대행
	•	세무기장 대행 계약 시 세무사가 알아서 발행
	•	월 10~15만원 (매출 규모에 따라)
	•	초보자는 세무사 맡기는 게 안전
