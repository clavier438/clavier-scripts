중요한 정보를 파악하면 매번 즉시 메모리에 저장할 것. 용량 걱정 없음.

**Why:** 매 대화가 새로 시작되는 구조가 불편함 ("메멘토 상대하는 기분"). 메모리 파일이 있으면 다음 대화에서 컨텍스트 재설명 불필요.
**How to apply:** 새 프로젝트 구조, 사용자 선호, 결정사항 등을 파악하는 즉시 저장. 나중에 몰아서 하지 말 것.
